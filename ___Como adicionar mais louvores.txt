1. Adicionar o hino à Listagem (dados.js)
Para o hino aparecer na tela inicial e na busca, você deve inseri-lo na categoria correta dentro do objeto coletanea.

Atenção: Lembre-se sempre da vírgula entre um hino e outro.

JavaScript

const coletanea = {
    "COLETANEA": [
        { num: "2201", titulo: "O Senhor é o meu pastor", tom: "C" },
        // Para adicionar o próximo, coloque uma vírgula no anterior:
        { num: "4405", titulo: "A minha alma anseia", tom: "G" },
        { num: "5500", titulo: "Maranata", tom: "A" } 
    ],
    "CIAs": [
        { num: "01", titulo: "Canta alegremente", tom: "D" }
    ]
};
2. Adicionar o Conteúdo (Letra e Acordes)
Atualmente, as letras estão dentro do arquivo cifra.html no objeto bancoDeCifras. Para cada hino que você criou no dados.js, deve haver uma entrada correspondente aqui.

Dica de Ouro: Para que o transpositor funcione, os acordes precisam estar entre as tags <b> e </b>.

JavaScript

const bancoDeCifras = {
    "2201": {
        titulo: "2201 - O Senhor é o meu pastor",
        corpo: `<b>C</b>             <b>G</b>
O Senhor é o meu pastor...`
    },
    // Adicionando um novo:
    "4405": {
        titulo: "4405 - A minha alma anseia",
        corpo: `<b>G</b>             <b>D</b>
A minha alma anseia por Ti...`
    }
};
3. Como formatar a cifra corretamente?
Para que a cifra fique alinhada, você deve usar espaços simples dentro da tag de texto pré-formatado (pre).

Dica Prática: Se você copiar a cifra de outro site, cole-a dentro das crases ( ` ) e depois coloque manualmente as tags <b> em volta dos acordes.

Exemplo: Se a cifra for C G, você deve escrever <b>C</b> <b>G</b>.

Dica para o Futuro: Arquivos Separados
Se você planeja colocar 500 ou 600 hinos, o arquivo cifra.html vai ficar muito grande. Quando chegarmos nesse nível, o ideal será criar um arquivo .txt ou .json para cada hino e fazer o site ler esse arquivo individualmente.

Mas, para começar com os hinos principais da sua igreja, o método acima é o mais rápido e fácil de testar.
